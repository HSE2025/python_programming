# Pyton HSE Course Materials

Materials for the introductory Python course (HSE). Each week pairs a Marp slide deck with a companion notebook so students can rehearse the lecture and then practice in Colab/Jupyter.

## Repository Layout
- `ipynb/` — week-by-week notebooks: переменные и имена, работа с текстом, списки и кортежи, циклы, функции и файлы, словари, модули и множества.
- `slides/` — Markdown decks (`Неделя 1.md` … `Неделя 7.md`) authored for [Marp](https://marp.app); export to PDF or present directly from VS Code.
- `instructions/` — setup aids such as `Python_Installation_Guide.ipynb` and the Week 9 cheat sheet on terminals, VS Code, and troubleshooting.
